How to Download

To obtain an executable copy of Slicer for the supported operating systems, please fill out the Software License Agreement.

When you submit this Agreement, we will check your submission. If all the fields are correctly filled out, you will then receive instructions via email explaining how to download the software. You should receive an email reply within a day, which will also include information on how to obtain the source code. If you do not receive a reply, please contact us at slicer@ai.mit.edu.

Compatibility

The software is currently available for the following platforms:
  • Sun UltraSPARC workstations running Sun Solaris 2.8 (also known as Solaris 8 or SunOS 5.8) and OpenGL 1.2
  • PCs running Windows 98, Windows NT, Windows 2000, or Windows XP.
  • Computers running Red Hat Linux 7.3.
  • Apple Macintosh computers running Mac OS X.
  • Other platforms (such as IRIX): we give out all source code, so users can compile it for their operating system.

Availability

Note: The feature supporting automatic registration by mutual information is not included in the current public release.
