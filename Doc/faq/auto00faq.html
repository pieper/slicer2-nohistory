• Where can I find example data to use with Slicer?

    You can download a tutorial file at https://share.spl.harvard.edu/share/birn/public/software/tutorial-with-dicom.tar.gz (Windows: https://share.spl.harvard.edu/share/birn/public/software/tutorial-with-dicom.zip).

    1. Download tutorial.tar.
    2. Uncompress tutorial.tar with WinZip, gunzip, or a similar utility. The tutorial.tar file contains the tutorial.xml file.
    3. Open the tutorial.xml file in Slicer. (Click File, then Open Scene, then select tutorial.xml.)
      The tutorial.xml file contains both volumetric data and 3D models.
