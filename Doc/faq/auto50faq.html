• Where is the source code if I have downloaded a binary distribution?

    The Tcl code is never compiled.
    You can find it in the http://www.na-mic.org/ViewVC/index.cgi/slicer2/Base/tcl/tcl-main/?root=Slicer2, http://www.na-mic.org/ViewVC/index.cgi/slicer2/Base/tcl/tcl-modules/?root=Slicer2, and http://www.na-mic.org/ViewVC/index.cgi/slicer2/Base/tcl/tcl-shared?root=Slicer2 directories.
    The C++ code is in slicer2/Base/cxx in the CVS repository, but it is not distributed with the binary packages.
