There are several ways to save where you are in Slicer.

  • Use Save Scene to save as an XML file. The XML file lists the elements of the scene and their attributes.
  • Use Save 3D View to save a picture of the 3D View or Save Active Slice to save one of the slices.
  • Use Save Current Options to save user preferences, like background color or view presets, into a special Options.xml file.
